# Smart Campus Resource Management System

This is the client-side application for the Smart Campus Resource Management System built using React. The application provides various functionalities for managing room bookings, inventory tracking, maintenance requests, and notifications.

## Features

- **Room Booking System**: Users can book rooms through a calendar interface, enter event details, and receive a QR code for access.
- **Inventory Tracker**: Displays available inventory items and allows users to request items.
- **Maintenance Requests**: Users can report maintenance issues and track their status in real-time.
- **Announcements and Notifications**: Admins can broadcast announcements, and users receive notifications for updates.
- **Data Analytics and Reports**: Visualizations for room usage and inventory trends, with options to download reports.
- **User Roles and Permissions**: Role-based access for students, faculty, admins, and IT staff.

## Installation

To run the client application, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the client directory:
   ```
   cd smart-campus-system/client
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the application:
   ```
   npm start
   ```
